# Password Strength Cracker

A Python-based cybersecurity project that evaluates password strength and identifies weak passwords based on security best practices.

## Features
- Checks password length and complexity
- Detects uppercase and lowercase letters
- Verifies numeric and special characters
- Identifies common weak password patterns
- Provides strength ratings (Weak, Medium, Strong)
- Gives recommendations for stronger passwords

## Technologies Used
- Python
- Regular Expressions (Regex)
- Tkinter (if GUI version)
- Cybersecurity Password Policies

## Purpose
This project helps users create secure passwords and understand password security principles, reducing the risk of unauthorized access and brute-force attacks.
